This directory contains coefficients for methods presented in the following paper paper:

Arash Sarshar, Steven Roberts, Adrian Sandu,
_Alternating directions implicit integration in a general linear method framework_,
Journal of Computational and Applied Mathematics,
2019,
112619,
ISSN 0377-0427,

https://doi.org/10.1016/j.cam.2019.112619.

The coefficents are written in Matlab syntax and are in rational form accurate up to at least 20 digits.
